About

What is this API for?

X-MEN API is a RESTful based on the animated serie homonymous. You can access to all characters, their powers, affiliation, etc, and the episodes presented in each season. Click on Doc seccion, and follow the steps to interact with the documentation.

Who are we?

We are Nicolas Bulgarella and Stefano Marcaccio . Like great geeks, we like superheroes and programming (...and a list of mandatory characteristics).

Why did we develop it?

It was a great opportunity for implement all the knowledge that we have gotten on our courses, and research those tools necessary for "tune up" an API. This is the first model. In the future, we would like to add and modify some characteristics on our website.

What did we implement to develop it?

We used Node, MongoDB, Heroku and Glitch in backend development. All requests return in JSON format. The hosting of the website is carry out through Netlify
